== Installation ==

You can install **It Slides** automatically from the plugin directory, or by uploading the files manually to your server. After activating **It Slides** you'll be forwarded to the settings section at the bottom of the menu **Settings > Reading**.

To uninstall, just deactivate and delete the plugin. Remember to remove the shortcodes you may have added using the plugin.
